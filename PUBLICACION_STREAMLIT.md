# Publicación rápida en Streamlit Community Cloud

1. Crea un repositorio en GitHub, por ejemplo `dashboard-bcrp`.
2. Sube todo el contenido de esta carpeta.
3. Entra a Streamlit Community Cloud y elige **New app**.
4. Selecciona el repositorio, la rama principal y el archivo `app.py`.
5. Publica la app y copia el enlace público.
6. Pega el enlace en la sección VI del documento final.

La aplicación funciona con la base local incluida. El botón **Actualizar desde API BCRP** intentará traer datos nuevos cuando la app tenga conexión a internet.
